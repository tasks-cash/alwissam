# Public Design System

## Personality

Premium dental clinic: calm navy + medical teal, high contrast, Arabic-first RTL with Source Sans 3 / Libre Baskerville for Latin.

## Tokens

| Token | Value |
| --- | --- |
| `--primary-navy` | `#0b2740` |
| `--medical-teal` | `#0d8f90` |
| `--soft-teal` | `#e5f6f5` |
| `--background` | `#f4f7fa` |
| `--surface` | `#ffffff` |
| `--radius` | `14px` |
| `--radius-lg` | `22px` |
| Fonts | Cairo (AR), Source Sans 3 + Libre Baskerville (LATIN) |

## Avoided

Purple gradients, glassmorphism overload, fake counters, rounded-full pill spam, low-contrast grey text.
